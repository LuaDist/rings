Overview

Rings is a library which provides a way to create new Lua States from within Lua. It also offers a simple way to communicate between the Master and the Slave states.

Rings is free software and uses the same license as Lua 5.0.

Status

Rings version 1.0.0 (for Lua 5.0) is now available for download.

Download

Rings can be downloaded in source code from the Lua Forge page. If you are using LuaBinaries Release 2 a Windows pre-compiled version of Rings can also be found at the same LuaForge page.
